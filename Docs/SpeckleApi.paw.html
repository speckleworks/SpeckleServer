POST - /auth/register

Description

Register a new user.

CURL

curl -X POST "http://localhost:8080/auth/register" \
    -H "Content-Type: application/json; charset=utf-8" \
    --data-raw "$body"

Header Parameters

  • Content-Type should respect the following schema:
{
  "type": "string",
  "enum": [
    "application/json; charset=utf-8"
  ],
  "default": "application/json; charset=utf-8"
}

Body Parameters

  • body should respect the following schema:
{
  "type": "string",
  "default": "{\"email\":\"johndoe@apple.com\",\"password\":\"luxurygoods\"}"
}

POST - /api/objects

Description

Creates and stores in the database a speckle object.

It will recursively look through the object’s properties and store any “heavy” geometries (Brep, Mesh, Curve and Polyline) in the Geometries database. All other geometries are stored within the object itself.

Response will contain the objects and geometries arrays of the saved documents.

    -H "Authorization: JWT eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJfaWQiOiI1OTUyOGFkNDE3NDFkZDU0MmViYjZlZmMiLCJuYW1lIjoiQW5vbnltb3VzIiwiaWF0IjoxNDk4NTgxNzE2LCJleHAiOjE1NjE2OTY5MTZ9.Py9Fwbuj-eiWJDBqZVRv8-uWsTChakhbrLcgKoMS0a4" \
